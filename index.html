Building a Website with Github in 10 Simple Steps

Ashley Kang | Last Updated Date.

1. Create a Github account

Go to github.com and create your account.

2. Create a repository

You can find the link here. You'll want to name your repository username.github.io. You will want to make your repository public, and tick of the "Add a README file" box. Your README file is where you will describe for other users what your program (website in this case) is about.

3. Install Github Desktop

You can find the download link here. Once you've installed GitHub Desktop, I recommend following the built-in tutorial that they provide you to get a general sense of how the program works.

4. Install Visual Studio Code (Or Your Text Editor of Choice)

You can find the download link for Visual Studio Code here. This will be where you write all of your code.

5. Open Your Github repository in Visual Studio Code

You can do this by going to "File", "Open recent", and selecting your GitHub repository.
Alternatively, you can go to "File", "Open Folder", and select your GitHub repository manually.

6. Add an index.html file

Click the “New File” button next to your repository name, and add an index.html file by typing it into the file name field. This will be where you build the "skeleton" of your website. A basic index.html setup would look something like this:

<!DOCTYPE html>
<html

<head
<meta charset="utf-8">
<meta name="viewport" content="width:device-width">
<title>My Website</title>
<link href="style.css" rel="stylesheet" type="text/css"/>
</head>
<body>
<h1>Hello World</h1>
</body>
<script src="script.js"></script>
</html>

7. Add a style.css file

This is where you will modify the appearance of your website.

8. Add a script.js file

This is where you can add some more intuitive and advanced elements to your website. JavaScript is not nessecary for a working website, but it will help the website feel more modern and user-friendly.

9. Configure your repository

This is most important step if you want yourself and others to be able to see your website. Go to the "Settings" tab at the far right. Go to the "Pages" tab under the "Code and Automation" category in the sidebar. In the "Source" dropdown menu, select "Deploy from a Branch". In the Branch dropdown menu, select the branch that you want your website to be hosted from. Typically, this will be the "main" branch.

10. Happy coding!
